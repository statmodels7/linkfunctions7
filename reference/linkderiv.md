# Evaluate Derivative of Link Function by Order

Routes to the correct forward derivative generic based on order.

## Usage

``` r
linkderiv(x, theta, order = 1)

linkderiv.link(x, theta, order = 1)
```

## Arguments

- x:

  An object of class `link`.

- theta:

  A numeric vector.

- order:

  An integer (0 to 4).
