# Evaluate Derivative of Inverse Link Function by Order

Routes to the correct inverse derivative generic based on order.

## Usage

``` r
linkinvderiv(x, eta, order = 1)

linkinvderiv.link(x, eta, order = 1)
```

## Arguments

- x:

  An object of class `link`.

- eta:

  A numeric vector.

- order:

  An integer (0 to 4).
